----------------------------------------------------------------
CONTENU DU PACK
----------------------------------------------------------------

1. email_relancer.py
   Envoie des emails de relance automatiques depuis un CSV.
   Deps : aucune (Python standard)

2. rapport_hebdo.py
   Génère un rapport HTML hebdomadaire et l'envoie par email.
   Deps : aucune (Python standard)

3. web_scraper.py
   Extrait données d'un site e-commerce vers CSV.
   Deps : pip install requests beautifulsoup4

4. linkedin_post_scheduler.py
   Publie des posts LinkedIn automatiquement via l'API.
   Deps : pip install requests

5. file_organizer.py
   Organise un dossier en sous-dossiers (par type/date/mot-clé).
   Deps : aucune (Python standard)

----------------------------------------------------------------
PRÉREQUIS
----------------------------------------------------------------

- Python 3.8 ou supérieur
- Installer les dépendances avec : pip install requests beautifulsoup4

----------------------------------------------------------------
DÉMARRAGE RAPIDE
----------------------------------------------------------------

1. Ouvre le script que tu veux utiliser dans un éditeur de texte
2. Modifie les variables dans la section CONFIGURATION en haut
3. Lance le script : python nom_du_script.py
4. Tous les scripts ont MODE_TEST = True par défaut — safe !
